# stock-tracker-project
## Overview

This is a simple Python project that calculates the total investment value based on user-input stock names and quantities. The program uses a predefined dictionary of stock prices and displays the total investment amount. It can also save the result to a `.txt` or `.csv` file.

## Features

* User-friendly input
* Predefined stock prices using a Python dictionary
* Calculates total investment value
* Displays investment summary
* Option to save results to a text or CSV file

## Technologies Used

* Python 3
* Dictionaries
* Loops
* Functions
* File Handling

## How to Run

1. Download or clone this repository.
2. Open the project folder in your preferred Python IDE or code editor.
3. Run the `stock_tracker.py` file.
4. Enter the stock name and quantity when prompted.

## Example

Available Stocks:

* AAPL - $180
* TSLA - $250
* GOOGL - $140
* MSFT - $420

Example Input:
Stock: AAPL
Quantity: 5

Output:
Total Investment = $900
